AI-Based Personal Styling & Fashion Trend Analysis System

This project is a machine learning-based fashion recommendation system that suggests personalized outfits based on user characteristics such as body type, skin tone, and occasion.

🎯 Problem Statement

Choosing the right outfit can be difficult due to varying body types, skin tones, and occasions. This project aims to simplify fashion decisions using data-driven recommendations.

💡 Solution

Developed an AI-based system that:
Recommends outfits tailored to user preferences
Uses machine learning for classification and personalization
Analyzes fashion trends using data visualization

⚙️ Tech Stack

Python
Pandas, NumPy
Matplotlib
Scikit-learn
SQLite (Database)

🧠 Features

Personalized outfit recommendations
User input based on body type, skin tone, and occasion
Trend analysis with visual charts
Structured and modular code using OOP principles
Data storage for user sessions

🚀 How to Run

pip install pandas numpy matplotlib scikit-learn
python main.py

📊 Output

Recommended outfits displayed in terminal
Trend visualization charts generated
User session stored in database

Add UI (web/app interface)
